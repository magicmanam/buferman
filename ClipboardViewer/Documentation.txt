	Документация:
Вы можете создать пару имя/пароль, используя контекстное меню.
При добавлении в буфер картинки, вместо тултипа добавляется превью картинки.
Неизвестные форматы записываются в файле с именем "unknown-formats.txt".
Чтобы отменить удаление/добавление буфера, используйте горячие клавиши Ctrl + Z, Ctrl + Y
Сразу же при запуске программы в буфере будет находится текущий клип (если он там есть).
Программа поддерживает русские и английские буферы.
Текст кнопки буфера показывает текст буфера с обрезанными лидирующими и заканчивающими знаками невидимых символов (пробелы, перевод каретки и т.д.).
Программа сохраняет буферы разных типов: любые тексты (plain text, html, Windows Office), файлы, снимки экрана и т.д.
Если при копировании в буфер приложение крешиться, опишите в каком приложении вы копируете и напишите на почту...
Для каждого буфера хранятся все его форматы, в которых он представлен. Форматы доступны через контекстное меню (правой кнопкой мыши).
Если буфер содержит картинку, то превью картинки можно увидеть в тултипе соответствующего буфера.
Программу желательно запускать в режиме Администратора, для корректной работы с приложениями, уже запущенными в этом режиме.
При выборе буфера, он поднимается вверх списка буферов, так как попадает в него снова.
Приложение позволяет сохранять буферы-файлы. Для удобства такие буферы обозначаются коричневым цветом.
В один момент времени на одном компьютере может быть запущено не более одного экземпляра программы. В панели задач пиктограмма программы отсутствует.
По нажатию на клавишу Escape, программа сворачивается.
Список доступных буферов появляется по нажатию на клавиши Alt+C. Навигироваться по буферам можно стрелками управления курсором, или клавишей Tab.
Для выбора нужного буфера, нужно нажать клавишу Enter или нажать левой кнопкой мыши. Буфер вставится туда, где стоит курсор активного окна.
Для удаления выделенного буфера нажмите кнопку Delete, или воспользуйтесь контекстным меню буфера.
При навигации по буферам клавишами управления курсора (или Tab), всплывает стандартный тултип.
После нажатия на Alt+C (или Alt+Tab), окно появляется снова, кнопки отрисовываются шириной, равной ширине окна программы.
При нажатии на кнопку закрытия окна программы или горячих клавиш Alt+F4 окно сворачивается, но не закрывается. Для закрытия программы используйте пункт меню Actions->Exit Session.
Возможность редактирования буферов отсутствует, ибо нефик. Каждый буфер хранится в нескольких форматах, в том числе бинарных, неподдающихся изменению.
Если вам нужно скрыть скопированный пароль, либо дать более смысловое имя-алиас для буфера, воспользуйтесь пунктом меню "Change text". Если изменённый таким образом буфер будет отличаться от первоначального значения, то будет выделен жирным шрифтом. Когда текст возвращается к первоначальному, шрифт становится обычным. Вернуть текст к первоначальному можно, воспользовавшись пунктом меню "Return text to initial", которое может доступно или недоступно.
Текст нельзя изменять для буферов с файлами или скриншотами.
Программа позволяет сохранять постоянные буферы (через пункт контекстного меню "Mark as persistent"), они не будут перезаписываться и всегда будут находиться внизу списка.
По нажатию на 'x' выделится первый буфер, а по нажатию на 'v' - последний.
