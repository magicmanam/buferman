Горячие клавиши

Alt+C - показать приложение

Esc or Alt+F4 - спрятать приложение

V (или End) - выделить самый нижний буфер, X (или Home) - выделить самый верхний буфер, TAB - выделить следующий буфер, C - как и TAB - 3 раза

Alt+P - поставить на паузу/возобновить перехват буферов

Ctrl+Z / Ctrl+Y - отмена/возврат предыдущей операции

Горячие клавиши буфера

Del - удалить буфер

Ctrl+C - поместить буфер в память без вставки

Ctrl+S - закрепить буфер

Ctrl+L - создать логин из буфера

Ctrl+H - скрыть текст буфера алиасом. Ctrl+I - удалить алиас

Документация

Интерфейс приложения имеет русскую и английскую версии. Переключать язык можно через главное менюю

Если в папке, где лежит программа, находится текстовый файл bufers.json, то при загрузке программы, буферы загрузятся автоматически. Изменить дефолтный файл можно через пункт главного меню.

Вы можете удалить все буферы, либо только временные.

Вы можете создать пару имя/пароль, используя контекстное меню.

При добавлении в буфер картинки, в качестве тултипа добавляется превью картинки.

Чтобы отменить удаление/добавление буфера, используйте горячие клавиши Ctrl+Z, Ctrl+Y. Эти же операции можно использовать для отмены других типов операций.

Сразу же при запуске программы в буфере будет находится текущий клип (если он там есть).

Программа поддерживает русские и английские текстовые буферы.

Текст кнопки буфера показывает текст буфера с обрезанными лидирующими и заканчивающими знаками невидимых символов (пробелы, перевод каретки и т.д.).

Программа сохраняет буферы разных типов: любой текст (обычный текст без форматирования, html, Microsoft Office), файлы, снимки экрана и т.д.

Если при копировании в буфер приложение крешится, опишите в каком приложении вы копируете и напишите на почту magicmanam@gmail.com

Для каждого буфера хранятся все его форматы, в которых он представлен. Форматы доступны через контекстное меню.

Программу желательно запускать в режиме Администратора, для корректной работы с приложениями, уже запущенными в этом режиме.

При выборе буфера, он поднимается вверх списка буферов, так как попадает в него снова.

Приложение позволяет сохранять буферы-файлы. Для удобства такие буферы помечаются иконкой файла.

В один момент времени на одном компьютере может быть запущено не более одного экземпляра программы. В панели задач пиктограмма программы отсутствует. Зато присутствует в системном трее. Трей имеет контекстное меню.

По нажатию на клавишу Escape, программа сворачивается, но не закрывается.

Список доступных буферов появляется по нажатию на клавиши Alt+C. Навигироваться по буферам можно стрелками управления курсором, или клавишей Tab. Также доступны горячие клавиши: 'X' - перейти в начало списка, 'C' - табулироваться на 3 буфера вперёд, 'V' - перейти в конец списка.

Для выбора нужного буфера, нужно нажать клавишу Enter или нажать левой кнопкой мыши. Буфер вставится туда, где стоит курсор активного окна.

Для удаления выделенного буфера нажмите кнопку Delete, или воспользуйтесь контекстным меню буфера.

При нажатии на кнопку закрытия окна программы или горячих клавиш Alt+F4 окно сворачивается, но не закрывается. Для закрытия программы используйте пункт главного меню или иконку в системном трее.

Если вам нужно скрыть скопированный пароль, либо дать более смысловое имя-алиас для буфера, воспользуйтесь контекстным меню. Вернуть текст к первоначальному можно, воспользовавшись контекстным меню или комбинацией клавиш Ctrl+I.

Программа позволяет сохранять постоянные буферы (через пункт контекстного меню), они не будут перезаписываться и всегда будут находиться внизу списка.

Постоянные буферы отделены от временных горизонтальной полоской.

Для удобства в системном трее имеется иконка программы.

Комбинация Alt+P при открытом окне программы - прекратить/восстановить перехват событий копирования. Удобно когда нужно вставить подряд несколько буферов и список буферов не должен меняться (для удобства выбора следующего).

Можно сделать любой буфер текущим с помощью Ctrl+C.
