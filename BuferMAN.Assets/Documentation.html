Hot keys

Alt+C - open BuferMAN instance

Esc or Alt+F4 - hide program

V (or End) - select the last bufer, X (or Home) - select the first bufer, TAB - select the next bufer, C - like TAB 3 times

Alt+P - pause/resume clipboard interception

Ctrl+Z / Ctrl+Y - undo / redo previous operation

Bufer hot keys

Del - delete bufer

Ctrl+C - place selected bufer in clipboard without pasting

Ctrl+S - pin bufer

Ctrl+L - create login from bufer

Ctrl+H - hide bufer text with some alias. Ctrl+I - remove alias

Documentation

There are two languages are supported: English and Russian. You can switch them via main menu.

If application folder contains file bufers.json, bufers from this file will be loaded on application launch automatically. You can change default file via main menu.

You can delete all bufers or only temporal.

You can create credentials from bufer using context menu or hot keys.

On image adding, there is a tooltip with image preview is added to the bufer.

In order to undo/redo any operation with bufers, use hot keys Ctrl+Z, Ctrl+Y

At application launch the program already contains current bufer if it exists.

BuferMAN supports text bufers in different languages.

Bufer's text trims leading and trailing whitespaces.

Application works with bufers of different types: any text (w/o formatting, html, Microsoft Office), files/folders, screenshots and so on.

If application is crashed, send an email with described application/conditions/steps to reproduce to magicmanam@gmail.com

All bufer's formats are stored and available for viewing - via context menu.

It's higly recommended to run BuferMAN in Administrator mode - for correct work with applications running in the same mode.

When you select any bufer, it bubbles up to the top of bufers list (as current one).

You can store/categorize bufers in files (txt or json). Bufers loaded from file are marked with file icon.

At one moment the only BuferMAN instance can be launched. There is no BuferMAN icon in task panel, but it exists in system tray.

Escape key hides the program (does not close).

You can see bufers list using hot key combination Alt+C. You can navigate in this list using arrow keys or Tab key. Also you can use key: 'X' - go to the top of the list, 'C' - 3 times Tab , 'V' - go to the bottom of the list.

To paste the bufer, click Enter key or use your mouse. Bufer will be pasted in active window.

To delete bufer click Delete key or use context menu.

Close button and Alt+F4 hot key combination hide the program, but do not close it. To close the program you need to use main menu or tray icon's context menu.

If you need to hide a password or give some alias to a bufer, use context menu or Ctrl+H hot key. Initial version can be returned via context menu or Ctrl+I hot key.

You can pin bufers (Ctrl+S) - they won't be rewritten by new bufers and will be located at the bottom of bufers list.

Pinned bufers are separated from usual bufers with horizontal line.

There is BuferMAN icon in system tray - use it to access Options window or when you need to close a program.

Alt+P hot key is used to pause/resume bufers intercepting. This is convenient when you need to paste several bufers and bufers list should store its order.

You can place any bufer in clipboard w/o pasting it via Ctrl+C hot key.
